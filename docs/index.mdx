# Welcome to Tixae Interviewer

Tixae Interviewer is a cutting-edge AI-powered interview platform that enables real-time voice conversations with intelligent AI agents. Built with modern web technologies, it provides a seamless interview experience available 24/7.

## Key Features

- **🤖 AI-Powered Interviews**: Experience genuine interviews conducted by advanced AI
- **🎙️ Voice Conversations**: Real-time voice calls with natural conversation flow
- **🔐 Secure Authentication**: Firebase-based authentication with Google sign-in
- **📊 Real-time Data**: Live conversation tracking and message history
- **🎨 Modern UI**: Beautiful, responsive interface built with Tailwind CSS and shadcn/ui
- **⚡ High Performance**: Built on Next.js 15 with Turbopack for lightning-fast development
